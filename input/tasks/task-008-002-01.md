# Task: Implement 4 portfolio tabs with 3 sub-tabs and default navigation

## Description
Build the portfolio tab bar with 4 horizontal tabs (Stock/Quality, Stock/Consensus, Options/Quality, Options/Consensus) and nest 3 sub-tabs (Signals, Positions, Performance) within each. Use bootstrap-vue-next `<b-tabs>` or equivalent. Default on page load is Stock/Quality + Signals. Clicking a portfolio tab preserves the current sub-tab. Signal type pills use blue for Quality, purple for Consensus.

## Acceptance Criteria
- [ ] 4 horizontal tabs rendered: Stock/Quality (default selected), Stock/Consensus, Options/Quality, Options/Consensus
- [ ] 3 sub-tabs per portfolio: Signals (default selected), Positions, Performance
- [ ] Tab switch preserves current sub-tab selection
- [ ] Page load always selects Stock/Quality + Signals (FR-050)
- [ ] Signal type color coding applied: blue for Quality tabs, purple for Consensus tabs
- [ ] Sub-tab content area renders placeholder text for each view

## Technical Notes
- Tabs map to portfolio parameters: Stock/Quality -> portfolio_id=1, signal_type=quality, instrument_type=stock
- Sub-tab routes: `/portfolio/:id/signals`, `/portfolio/:id/positions`, `/portfolio/:id/performance`
- Use bootstrap-vue-next `<b-tabs>` and `<b-tab>` components
- Tab switching should update URL via router to enable bookmark-like behavior (though no persistence)
